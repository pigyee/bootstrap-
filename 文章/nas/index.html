把移动送的魔百盒魔改成nas


2024.1.17
阅读量:

魔百盒，不只是电视。下次教大家用魔百盒+内网穿透搭网站

前提条件

盒子需要安装第三方软件，不刷机安装第三方软件可以看这篇教程co↑co↓如果没有可以用的方法，可以刷机或者连接TTL。杀鸡有风险，刷机需谨慎

下载ksweb

ksweb内置了FTP服务器，Web Interface可以在浏览器里管理ksweb的设置。前三个事服务器软件，跑网页的

  • lighttpd server
  • nginx
  • Apache
  • PHP
  • MySQL
  • msmtp
  • Web Interface
  • KSWEBFTP
  • Editor
  • Scheduler

下载链接按照前提条件里提到的安装第三方软件的方法安装ksweb，有的盒子是静默安装，有的盒子会提示，安装即可。

配置ksweb

如果遥控器操作不方便，可以连接USB鼠标或无线鼠标，这里用平板电脑演示

打开之后差不多是这样的，把不用的服务都关掉，只留FTP

点开ftp的选项，其他的不用管点下面的加号，添加一位用户

用户名密码随便填，下面的允许修改文件勾上。如果插了U盘，把密码下面的储存位置改成u盘位置，可以节省盒子空间，把文件存到u盘上

已经快完成了，还差最后一步

按图片操作，在盒子启动时就可以自动启动ksweb

连接FTP服务

恭喜，服务端已经配置完成了，用文件管理器连接盒子即可

连接要填的
协议 FTP
服务器地址 从路由器管理页面查看或者在盒子网络设置页面查看
端口 默认为2121
路径 不填
服务器地址 从路由器管理页面查看或者在盒子网络设置页面查看
用户名密码 在ksweb上填的
编码 UTF-8
传输模式 被动

效果

速度飕飕的，像过山车一样

适合存最新的电影，文件管理器是solidexplorer
